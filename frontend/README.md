# Система мониторинга цен на электроэнергию

Это приложение для управления базой данных цен и потребления электроэнергии в РФ (Европейская и Азиатская части).

## Структура проекта
- `backend/` — серверная часть на FastAPI.
- `frontend/` — пользовательский интерфейс на Streamlit.
- `data.csv` — локальная база данных.
- `requirements.txt` — список необходимых библиотек.

## Как запустить

1. Установите библиотеки:
   pip install -r requirements.txt

2. Запустите Бэкенд (из корня проекта):
   uvicorn backend.main:app --reload

3. Запустите Фронтенд (в новом окне терминала):
   streamlit run frontend/app.py

## Функционал
- Просмотр таблицы данных.
- Интерактивный график Plotly и статический график Streamlit.
- Форма для добавления новых записей.

- Удаление записей по ID.
